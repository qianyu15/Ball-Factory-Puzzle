# ⚙️ Ball Factory Puzzle

物体や装置を配置して、物理演算で動くボールをゴールまで導くブラウザパズルゲーム。

坂道、壁、バネ、回転ゲートなどを組み合わせて、ボールがゴールへ到達するルートを作ろう。

## 🎮 Demo

ブラウザでHTMLファイルを開くだけでプレイできます。

## ✨ Features

* 🎯 ボールをゴールへ導く物理パズル
* 🟦 坂道
* 🧱 壁
* 🟡 バネ
* 🌀 回転ゲート
* 🗑 オブジェクト削除
* ⚙️ 重力・摩擦・反発を利用した簡易物理演算
* 📱 Pointer Events対応
* 💻 PC・スマートフォン・タブレットでプレイ可能
* 🎨 Tailwind CSSを使用したUI
* 🧩 Canvasによるゲーム描画

## 🕹️ How to Play

### 1. オブジェクトを選択

画面上部のツールバーから配置するオブジェクトを選択します。

| オブジェクト   | 説明                   |
| -------- | -------------------- |
| 🟦 坂道    | ボールを滑らせる             |
| 🧱 壁     | ボールを跳ね返す             |
| 🟡 バネ    | ボールを強く弾き飛ばす          |
| 🌀 回転ゲート | 回転するアームでボールの進行方向を変える |
| 🗑 削除    | 配置したオブジェクトを削除する      |

### 2. フィールドに配置

選択したツールでキャンバスをクリック、またはタップするとオブジェクトが配置されます。

### 3. スタート

**▶ スタート**ボタンを押すとボールが動き始めます。

重力や各オブジェクトの効果を利用して、ボールをゴールまで導きます。

### 4. ゴール

ボールが緑色のゴールに到達するとゲームクリアです。

## 🔧 Game Mechanics

### Gravity

ボールには常に重力が働きます。

```js
const gravity = 0.45;
```

### Friction

移動中のボールには摩擦による減速が発生します。

```js
const friction = 0.985;
```

### Ball Physics

ボールは以下の要素によって動きます。

* 重力
* 水平方向の速度
* 垂直方向の速度
* 摩擦
* 画面端との衝突
* オブジェクトとの衝突
* 反発

## 🧩 Objects

### 🟦 Ramp

ボールを坂道の方向へ滑らせます。

```js
{
    type: "ramp",
    x,
    y,
    w: 180,
    h: 25,
    angle: -0.35
}
```

### 🧱 Wall

ボールを反射させる障害物です。

```js
{
    type: "wall",
    x,
    y,
    w: 150,
    h: 25,
    angle: 0
}
```

### 🟡 Spring

ボールが接触すると、外側へ強く弾き飛ばします。

```js
{
    type: "spring",
    x,
    y,
    r: 28
}
```

### 🌀 Spinner

3本のアームが回転し、ボールに力を加えます。

```js
{
    type: "spinner",
    x,
    y,
    r: 60,
    angle: 0
}
```

## 🚀 Run Locally

リポジトリをクローンします。

```bash
git clone https://github.com/your-username/ball-factory-puzzle.git
```

プロジェクトディレクトリへ移動します。

```bash
cd ball-factory-puzzle
```

その後、`index.html`をブラウザで開きます。

ローカルサーバーを使用する場合は、例えば以下のように実行できます。

```bash
python -m http.server 8000
```

ブラウザで以下を開きます。

```text
http://localhost:8000
```

## 🧠 Goal

限られた配置スペースの中で装置を組み合わせ、ボールをゴールまで導くことが目的です。

坂道で加速させ、壁で方向を変え、バネで飛ばし、回転ゲートでさらに進行方向を変える。

つまり、ボールに「自力でゴールへ行け」と言うのではなく、プレイヤーが物理法則を利用して道を作るゲームです。
